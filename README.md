## Jarir-Pharmacy

Jarir-Pharmacy is App For Request Medicines From Pharmacies and Know the Prices of the Products with Different Departments and Can Buy it Online

## Features
- [x] Kotlin
- [x] Kotlin Coroutines
- [x] Navigation Component
- [x] MVVM
- [x] LiveData
- [x] Hilt
- [x] SOAP
- [x] Proguard Rules
